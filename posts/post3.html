Professional Documentation: Installing Ruby and Rails with rbenv on Ubuntu

Overview

This guide provides step-by-step instructions for installing Ruby, Rails, and essential dependencies on Ubuntu using rbenv. This setup ensures an isolated and flexible Ruby environment for development.

Prerequisites

  • A system running Ubuntu (24.04 or later recommended)
  • Sudo privileges
  • Basic familiarity with the command line

Step 1: Update System Packages

Before installation, update and upgrade your system packages:

sudo apt update && sudo apt upgrade -y

Step 2: Install Required Dependencies

Run the following command to install essential packages:

sudo apt install -y curl git gcc make libssl-dev libreadline-dev zlib1g-dev libsqlite3-dev
    sudo apt install -y libffi-dev libyaml-dev

Step 3: Install rbenv

Clone the rbenv repository and update system path:

git clone https://github.com/rbenv/rbenv.git ~/.rbenv

Add rbenv to your shell by running:

echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.bashrc
    echo 'eval "$(rbenv init - bash)"' >> ~/.bashrc
    source ~/.bashrc

Step 4: Install ruby-build Plugin

git clone https://github.com/rbenv/ruby-build.git ~/.rbenv/plugins/ruby-build

Verify rbenv installation:

rbenv --version

Step 5: Install Ruby

List available Ruby versions:

rbenv install -l

Install the latest stable version of Ruby (replace 3.X.X with the latest version):

rbenv install 3.X.X
    rbenv global 3.X.X

Verify the installation:

ruby -v

Step 6: Install Bundler and Rails

gem install bundler
    gem install rails -v 8.0.1 # Install the latest Rails version
    rbenv rehash

Verify Rails installation:

rails -v

Step 7: Install Database Dependencies

For PostgreSQL:

sudo apt install -y postgresql postgresql-contrib libpq-dev

For SQLite:

sudo apt install -y sqlite3 libsqlite3-dev

Conclusion

You have successfully installed Ruby, Rails, and essential dependencies using rbenv on Ubuntu. You can now start developing Ruby on Rails applications.
